# 08-vertexai-workbench

**WHAT:** A managed Vertex AI Workbench (Jupyter) instance attached to our VPC.

**WHY:** For interactive EDA, debugging, and notebook-based prototyping. The notebooks under `src/notebooks/` can be opened here with one click.

**HOW:** Single `google_workbench_instance` (gen2) with a small machine type to keep cost low.

**JD KEYWORD:** Vertex AI Workbench
